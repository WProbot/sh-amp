# amp   
Easy and fast installation minimum amp stack (apache2 + mariadb10 + php7 + vsftpd + sendmail).   

## Installation   
```
$ sudo su
$ git clone https://github.com/w3src/sh-amp.git
```

## Usage   
```
$ cd sh-amp
$ chmod +x ./ubuntu18.04-amp.sh
$ ./ubuntu18.04-amp.sh
```
