# amp   
Easy and fast installation of the latest version of the amp stack (apache2 + mariadb10 + php7 + vsftpd + sendmail).   

## Install   
```
$ sudo su
$ git clone https://github.com/w3src/sh-amp.git
$ cd sh-amp
$ chmod +x ./*.sh
$ ./install.sh
```

## Update   
```
$ ./update.sh
```
